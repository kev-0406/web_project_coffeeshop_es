# Triple Espress - Pagina web de una Cafeteria.

## Descripcion del proyecto

Este proyecto es una pagina web estatica (Landing Page) para una cafeteria de especialidad, la principal funcion es presentar el negocio, sus horarios de atencion y ofreces una interfaz intuitiva para que los usuarios puedan reservar una mesa.

## Tecnologias utilizadas:\*\*

- HTML5 Semantico
- CSS3 Puro (Flexbox, Posicionamiento Absoluto/Relativo).
- Metodologia BEM para una arquitectura de codigo organizada.
- Transiciones y pseudo-clases de CSS para interactividad.

## Planes de mejora del proyecto

A medida que continúe trabajando en este proyecto, me gustaría implementar las siguientes funciones para mejorar la experiencia del usuario y hacer la página más completa:

- **Menú y Galería de Productos:** Añadir una nueva sección visual que muestre fotografías de los cafés de especialidad que se manejan, acompañadas de una breve descripción de sus notas de sabor y origen.
- **Sección de Métodos de Extracción:** Crear un bloque informativo donde se expliquen los diferentes métodos de preparación que ofrece la cafetería.
- **Diseño Responsivo (Adaptabilidad Móvil):** Ajustar la estructura mediante CSS para que tanto las nuevas galerías visuales como el formulario de reservas se adapten y funcionen perfectamente en las pantallas de los teléfonos celulares.
